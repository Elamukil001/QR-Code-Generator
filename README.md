# QR Code Generator Setup and Run Instructions

This is a Python Flask application. Follow these steps to run it.

## Prerequisites

1.  **Install Python**: You need Python installed on your system.
    *   Download it from [python.org](https://www.python.org/downloads/).
    *   **Important**: During installation, check the box that says **"Add Python to PATH"**.

## Setup

1.  Open a terminal (Command Prompt or PowerShell) in this folder.
2.  Install the required libraries:
    ```bash
    pip install -r requirements.txt
    ```

## Running the Application

1.  Run the application with:
    ```bash
    python app.py
    ```
2.  Once running, open your browser and go to:
    [http://127.0.0.1:5000](http://127.0.0.1:5000)

## Troubleshooting

*   If `python` is not recognized, try using `py` or `python3` instead.
*   If you see "pip is not recognized", ensure Python was added to your PATH during installation.
